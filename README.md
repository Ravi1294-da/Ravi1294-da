<summary><b>🏥 Medicare Claims Analytics Platform</b></summary>

### Portfolio Project

An end-to-end healthcare analytics project built using **Python, Google BigQuery, dbt, SQL, and Tableau Public** to transform CMS DE-SynPUF Medicare claims data into a cloud-based analytics solution and executive dashboard.

#### Highlights

* Built Python data cleaning and feature engineering pipelines using Pandas
* Loaded and validated Medicare claims data into Google BigQuery
* Developed dbt staging models, fact models, and data quality tests
* Built an interactive Tableau dashboard featuring KPIs, geographic analysis, demographic insights, and healthcare cost reporting
* Analyzed **857,549 Medicare claims** across inpatient and outpatient services
* Documented data quality findings and business insights throughout the pipeline

#### Key Results

| Metric                 |         Value |
| ---------------------- | ------------: |
| Total Claims           |   **857,549** |
| Total Claim Spend      |   **$863.8M** |
| Average Claim Cost     |    **$1,007** |
| Average Length of Stay | **5.72 Days** |
| CHF Cost Difference    |    **+35.1%** |

#### Technologies

`Python` `Pandas` `Google BigQuery` `dbt` `SQL` `Tableau Public` `Git`
